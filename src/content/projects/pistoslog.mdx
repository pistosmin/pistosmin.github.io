pistoslog는 개인 블로그와 포트폴리오 사이에 있는 공간이다. 글을 읽는 경험을 우선하면서, 이력서나 GitHub 프로필에서 링크했을 때 작업물의 맥락도 함께 보여주는 것을 목표로 한다.

## 목표

- 일상, 생각, 기술 공부, 작업 기록을 분리해서 관리한다.
- MDX 기반으로 글을 쉽게 추가한다.
- GitHub Pages에 정적으로 배포한다.
- 댓글, 공감, 검색, RSS, SEO를 기본 구조에 포함한다.

## 디자인 방향

토스처럼 명확하고, 카카오처럼 친근하며, Medium처럼 읽기 편하고, velog처럼 개발 기록을 탐색하기 쉬운 형태를 지향한다. 리퀴드 글래스는 핵심 인터랙션 주변에만 제한적으로 사용한다.
