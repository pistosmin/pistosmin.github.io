gravity는 모바일 퍼스트 커뮤니티 웹앱을 만들며 프론트엔드, 백엔드, 데이터베이스, 캐시, 로컬 인프라를 한 흐름으로 묶어본 실험이다.

핵심은 게시판 기능 자체보다 개발 루프였다. 인증, 게시글, 댓글, 추천 같은 익숙한 기능을 바탕으로 React 클라이언트와 Spring Boot API, PostgreSQL, Redis, Docker Compose가 어떻게 맞물리는지 확인했다.

이 프로젝트는 이후 Claude Code와 서브에이전트를 나눠 쓰는 방식으로 커졌다. 계획, 구현, 리뷰, 검증을 분리하면서 AI가 코드를 얼마나 빨리 만들 수 있는지보다 사람이 어느 지점에서 판단해야 하는지를 배운 작업이었다.

공개 저장소는 당시 실험이 남아 있는 `base-one` 이름으로 연결되어 있다. 프로젝트 카드에서는 블로그에서 정리한 작업명인 `gravity`를 사용했다.
